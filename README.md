# taotao-parent
Taotao商城练手项目
项目来源于网络资源
#   技术架构为：
##	Spring、SpringMVC、Mybatis
## JSP、JSTL、jQuery、EasyUI、KindEditor（富文本编辑器）
##	Redis（缓存服务器，单点登录，购物车）
##	Solr（搜索）
##	dubbo（分布式服务框架）
## HttpClient（HTTP 协议访问客户端）
##	ActiveMQ（消息队列）
##	Quartz（定时任务）
##	FastDFS（图片服务器）
##	FreeMarker（网页静态化）
## Nginx（反向代理服务器）
## MyCat（数据库中间件）
